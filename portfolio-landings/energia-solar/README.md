# Solarys Energia

Landing page profissional para o nicho **Empresa de Energia Solar**, criada para compor o portfólio de Juliana Rainen.

## Objetivo

Apresentar uma empresa fictícia com visual comercial, foco em confiança e incentivo ao contato pelo WhatsApp.

## Tecnologias

- HTML5
- CSS3
- JavaScript puro

## Funcionalidades

- Layout responsivo
- Menu mobile
- Scroll suave
- Animações leves ao rolar a página
- Botão voltar ao topo
- Botão de WhatsApp
- Formulário com validação
- SEO, Open Graph e favicon

## Como executar

Abra o arquivo `index.html` no navegador ou rode um servidor local na pasta do projeto.

## Estrutura de pastas

```
energia-solar/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    ├── images/
    └── icons/
```
